## 🔍 快速查找替换方法

使用编辑器批量替换：

```bash
# 在所有 .mdx 文件中查找并替换
# 方法 1：使用 sed (macOS/Linux)
find content/pages -name "*.mdx" -exec sed -i '' 's/YourAppName/你的应用名/g' {} +
find content/pages -name "*.mdx" -exec sed -i '' 's/your-domain\.com/你的域名.com/g' {} +
find content/pages -name "*.mdx" -exec sed -i '' 's/support@your-domain\.com/support@你的域名.com/g' {} +

# 方法 2：在 VSCode 中使用查找替换
# Cmd/Ctrl + Shift + H 打开全局查找替换
# 搜索：YourAppName → 替换为：你的应用名
# 搜索：your-domain.com → 替换为：你的域名
# 搜索：support@your-domain.com → 替换为：你的支持邮箱
```
