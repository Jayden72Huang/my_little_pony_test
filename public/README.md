## Image Guidelines

- Use PNG format with transparency for best quality
- Keep file sizes under 500KB per image (optimize with tools like TinyPNG)
- Ensure images are high-quality and properly cropped
- Character images should show the character's face/full body clearly
- Logo should work on both light and dark backgrounds
